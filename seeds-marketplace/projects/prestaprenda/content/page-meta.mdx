# PrestaPrenda · Page Meta Seeds

Lorem ipsum dolor sit amet, consectetur adipiscing elit.  
This page exposes all **content seeds** (content_instances + print_versions) that use the **Page Meta** template for PrestaPrenda.  

It is intended as a **copy-paste repository** for seeding multiple records.

---

## 📋 How to use

- Each record consists of:
  - A `content_instances` object
  - Its related `print_versions` object
- All records follow the **Page Meta template**.
- You can paste multiple records into the array below.
