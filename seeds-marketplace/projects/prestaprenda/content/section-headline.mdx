# PrestaPrenda · Section Headline Seeds

Lorem ipsum dolor sit amet, consectetur adipiscing elit.\
This page exposes all **content seeds** (content_instances \+ print_versions) that use the **Section Headline** template for PrestaPrenda.

It is intended as a **copy-paste repository** for seeding multiple records.

---

## 📋 How to use

- Each record consists of:
  - A `content_instances` object
  - Its related `print_versions` object
- All records follow the **Section Headline template**.
- You can paste multiple records into the array below.
