| Field                        | Description                                 |

| **Default Meta Description** | Fallback meta description                   |
| **Canonical Base URL**       | Base URL for canonical tags                 |

| **Robots Follow**            | Allow/disallow link following               |
| **Open Graph Image Default** | Default OG image                            |
| **Social Profiles**          | Links to social profiles                    |
| **Share Hashtags**           | Default hashtags for sharing                |
